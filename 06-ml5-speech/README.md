# Einführung in ML 6: Speech Recognition with ML5js

In diesem Experiment erkennen wir bestimmte Befehle mithilfe von [ML5JS Speech Recognition](https://ml5js.org/reference/api-soundClassifier/).

## Installation

ML5js-Experimente werden über einen Webserver zur Verfügung gestellt.
Hierfür muss ein Webserver installieren werden. Dies geschieht wie gewohnt über NPM:

```bash
npm install
```

## Das Experiment starten

Das Experiment starten wir nun wie gewohnt über npm:

```bash
cd C:\Projekt\Ordner
npm start
```

Danach ist die Übung über einen Webbrowser der Wahl unter folgender Addresse erreichbar:
[http://localhost:1234](http://localhost:1234)
